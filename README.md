## My Contribution

This is a fork of a team mini-project. My individual contribution was focused 
on the **frontend**:

- Built reusable React UI components used across Student, Teacher, and 
  Administrator dashboards
- Implemented UI views for course management, assignment submission, and 
  attendance tracking
- Connected frontend components to backend API endpoints to test and 
  validate UI functionality during development

Backend, database design, and other modules were built by teammates as part 
of this collaborative project.
